# `icp_ledger_agent`

An AI agent example to interact with ICP blockchain ledgers.

## Running locally

```sh
git clone https://github.com/ldclabs/anda.git
cd anda
mkdir -p object_store
cp example.env .env
# update .env
cargo run -p icp_ledger_agent
```
